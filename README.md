# eldION - Prosjektinstrukser for Claude

Prosjekt: Modernisering av eldion.no. Selskapet leverer trådløse, batteriløse adapterer til insdustrielle sensorer/måleinstrument drevet av spillvarme til industrien.

Teknologistakk:

Rammeverk: Astro (Static Site Generation)

Styling: Tailwind CSS

Backend (Senere fase): Java Spring Boot

Designprinsipper:

Stil: Profesjonelt, industrielt og moderne. Bruk eksisterende "glassmorphism"-effekter, men gjør dem renere med Tailwind. Stilige animasjoner som får nettsiden til å føles moderne.

Farger: Bevar eksisterende fargepalett (cyan/blå aksenter som hovedfarge, med oransje som kontrast).

Kodekvalitet: Skriv ren, modulær kode. Del UI inn i små, gjenbrukbare komponenter (f.eks. <Navbar/>, <TeamCard/>).

Arbeidsmetodikk for Claude:

Gjør endringer iterativt.

Spør om tillatelse før du sletter store mengder opprinnelig innhold.

Sikre at nettsiden er 100 % responsiv på mobil og PC.
